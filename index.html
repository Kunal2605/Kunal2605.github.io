Hospital Application

The purpose of Appointment Scheduler & Manager application is to automatically schedule and manage patient’s appointments with a doctor of an organization by accessing their calendar in the user requested time and duration. This application would help both the doctors and patients to save their time. Managing delays and priority appointments is the primary aim of the application. This application is basically implemented in two interfaces: patient interface and doctor interface. Two main features of the application are scheduling and managing.
